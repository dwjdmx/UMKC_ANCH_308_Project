Whitespace is the term used to describe the areas of the page WITHOUT text. A few examples of whitespace are the space characters between words and the indentation at the beginning of a line.
Sometimes it can be useful to visualize whitespace. To do this click on the pilcrow icon on the home tab.
Here we can see the visualization of whitespace in our document. Word visualizes spaces as dots, new lines as pilcrows, and indentation as arrows.

As we can see on the third line, there is no arrow indicating the indentation. This because that line was not indented using the tab key, but instead using the preferred method of ruler indentation.
The preferred method of indentation is to use the ruler. To adjust indentation using the ruler simply select and drag the hourglass shape the left or right while having the desired paragraph selected.
Another type of useful whitespace is the page break, which can be inserted by holding the control key and pressing enter. A page break makes sure that everything coming after it start on a new page. This is useful in many occasions. At the end of a chapter for example.
