Styles are types of formatting that are applied on the paragraph level.
To modify an existing style right click it and select modify. In this case we will modify the “Normal” style which is the default style for paragraphs.
Here we can find many options we are already familiar with such as font, size, and alignment.
More paragraph options can be found by going to format and then selecting paragraph.
Under general we can adjust alignment.
Under indentation we can adjust the indentation such as indent the first line of each paragraph.
And under spacing we can change the spacing between paragraphs and between lines within the same paragraph.
