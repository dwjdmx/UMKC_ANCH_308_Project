Text consists of a few key attributes: font, size, color, and modifications.
Before we can change any of these aspects of text we must select the text we wish to change.
The font is the overall look of the text. Is it strait? Is it curly? Is it cursive? Does it have serifs? These are the broad stylistic attributes that are controlled by the font. The font can be changed by selecting one from the font dropdown list in the top left of the home tab.
For this example we chose the ever popular “Times New Roman.”
Similarly we can change the size of the text by selecting a size from the size dropdown menu in the top left of the home tab.
To change the color of the text we can click on the “A” icon and choose from a number of swatches, or select “More Colors…” to choose a custom color. We will choose “Automatic” to use the recommended default color.
There are also a number of other ways we can modify the visuals of the text. To bold the selected text click the “B” icon.
To italicize the text select the slanted “I” icon.
And finally, to underline the text selected the “U” icon.
The horizontal position of the text can be adjusted to be right aligned…
… center aligned…
… or left aligned.
